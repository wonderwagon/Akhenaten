I wasn't sure whether to create 4 separate threads for each game or just keep them all together as they seem to share some file formats.

Caesar 3 (1998)
Pharaoh (1999)
Zeus (2000)
Emperor (2002)

All games for PC Windows
2D graphics / city building sim genre
See Mobygames entries here

The main files that all 4 games share seem to be PAK & 555
Also SG3, M3D, EMP & ENG which may need to be extracted also.

555, SG2 & SG3 appear to be graphic/image files.

List of file formats for each game:
Caesar 3 - pak, emp, 555, sg2, eng, smk, map
Pharaoh - pak, emp, 555, sg3, m3d, map
Zeus - dat, pak, 555, sg3, m3d, eng, set, map
Emperor - pak, 555, sg3, m3d, eng, cfg, map

Download a sampling of files from each game HERE.

I see PAK files on the list of games here so maybe there will be some news for these games.

Some players have tried hex editing to change the PAK files which you can read about here.
There is also a utility for SG2 & SG3 extraction here and here.

And an Open Source Caesar 3 project here.

Thanks for your time!
